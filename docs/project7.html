Automated X-Ray Measurement

Computer Vision Machine Learning Healthcare Rothman Orthopaedic Institute

In partnership with the Rothman Orthopaedic Institute, I am beginning a project focused on automating the measurement process of post-operative X-rays for total ankle replacement surgeries.

Currently, the measurement process for evaluating the alignment and positioning of ankle implants is performed manually by clinicians, which can be time-consuming and subject to variability. This project aims to leverage computer vision and machine learning techniques to automate these measurements, improving both efficiency and consistency in post-operative assessment.

This project is currently in its early stages. More details will be shared as the work progresses.
